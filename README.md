# CLEAR - Classroom Learning Evaluation and Assessment Repository

CLEAR is a web-based Quality Education Management System designed to support effective classroom assessment and learning evaluation.

## Features

### Teacher Module
- Teacher Authentication
- Create Learning Activities
- Quiz Builder
- Seminar Assessments
- Student Response Management
- Performance Reports
- OBE Analysis Dashboard

### Student Module
- Student Authentication
- View Assigned Activities
- Attend Assessments
- Submit Responses
- View Scores
- Feedback Diagnostics

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Local Storage Database

## Project Objective
To provide a centralized platform for teachers to assess learning outcomes and for students to participate in interactive classroom evaluations.
